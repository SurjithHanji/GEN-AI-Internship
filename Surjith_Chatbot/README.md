# NexusAI — Chat UI

**CampusPe Gen AI Assignment** | Student Chat UI Project

---

## Overview

NexusAI is a modern, responsive chat user interface inspired by platforms like ChatGPT and Claude.  
It is built using HTML, CSS, JavaScript, jQuery, and Bootstrap 5 with no backend — focusing purely on UI/UX and interactivity.

This project demonstrates responsive design, smooth animations, and dynamic chat handling with a clean and professional interface.

---

## Features

### Core Tasks (100 pts)

- ✅ **Task 1 — HTML Structure**
  - Semantic HTML5 layout (header, main, section)
  - Welcome screen with suggestion cards
  - Input area with textarea and buttons
  - Typing indicator structure

- ✅ **Task 2 — CSS Styling**
  - CSS variables for colors, spacing, and transitions
  - User & AI message bubbles (left/right aligned)
  - Smooth animations and hover effects
  - Responsive layout using Flexbox & Grid

- ✅ **Task 3 — JavaScript & jQuery**
  - `addMessage()` and `sendMessage()` functions
  - Enter key handling (Shift+Enter for new line)
  - Mock AI responses with delay
  - Typing indicator animation
  - Auto-scroll functionality

- ✅ **Task 4 — Sidebar & Responsive Design**
  - Fixed 260px sidebar with chat history
  - New Chat button (fully functional)
  - Clickable chat switching
  - Mobile hamburger menu
  - Fully responsive from 320px to 1920px
